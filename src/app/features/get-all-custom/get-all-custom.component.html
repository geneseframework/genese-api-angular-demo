getAllCustom(path: string, params?: GetAllParams): Observable< T[] >

This method is used to get a list of objects from http GET request and to format them with T type.

Contrary to getAll() method, you can use a custom api path and you don't need to parameter your model with a genese property.

The backend must wait a GET http request.

In this example, the getAllCustom() method receives the list of books sent by the NestJS backend.

Open your navigator console. In the tab "Network", you'll find the data received from GET request. In the tab "Console", you'll can compare these original data with the formatted objects sent as Observable by Genese. As you can see, Genese formatted all these objects with "Book" type.
