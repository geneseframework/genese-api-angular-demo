deleteCustom(path: string, id?: string, options?: RequestOptions): Observable< ResponseStatus >

deleteCustom() method is used to removed a T object in database with a DELETE http request and returns a response with ResponseStatus type.

Contrary to delete() method, you can use a custom api path and you don't need to parameter your model with a genese property.

However, the backend must wait a DELETE http request.

In this example, the deleteCustom() method removes a book with the help of the NestJS backend.

Open your navigator console. In the tab "Network", you'll find the data received from DELETE request.
